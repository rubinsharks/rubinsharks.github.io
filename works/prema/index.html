Prema

Simple html site generator!
디렉토리 형태의 TOML 레이아웃 및 마크다운을 홈페이지 형태의 html로 생성해줍니다.
하나의 디렉토리엔 하나의 index.toml 또는 index.md가 메인 레이아웃이 됩니다. 또한 디렉토리는 URL을 의미하기에 portfolio디렉토리는 {homepage url}/portfolio가 됩니다.

Github

https://github.com/rubinsharks/pretty_mark

Execution 

						prema html {source_folder} {target_folder}

Source Folder Structure Example 

						.
├── about me
│   └── index.md
├── portfolio
│   ├── index.toml
│   ├── android.md
│   ├── ios.md
│   └── ai
│       ├── index.toml
│       └── cnn.md
├── index.toml
└── logo.svg
